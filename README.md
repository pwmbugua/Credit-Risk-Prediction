#Credit Risk Prediction using Machine Learning

A data science project that analyzes applicant information and builds a predictive model to determine loan approval status. The project involves data cleaning, categorical encoding, and machine learning using Python and scikit-learn.

#Project Overview
This project builds a machine learning model to predict whether a loan application should be approved or rejected based on applicant data.

#Objective
To assist financial institutions in making data-driven decisions when evaluating loan applications.

#Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn

#Process
1. Data loading and exploration  
2. Handling missing values  
3. Encoding categorical variables  
4. Splitting data  
5. Model training (Logistic Regression)  
6. Model evaluation  

#Results
- Accuracy: 78.9%
- Strong prediction for approved loans  
- Lower recall for rejected loans
